Marathon Web : 20 Décembre 2018 - 23 Décembre 2018

Projet réalisé à l'IUT de Lens
Description
Projet marathon du web 2018 : projet réalisé au sein d'une équipe composée d'étudiants en DUT Informatique et DUT MMI, nous avons réalisé un site web en collaboration, avec d'un côté des développeurs BackEnd et de l'autre les développeurs FrontEnd.
Objectif
L'objectif de ce projet "marathon" était de réaliser en 34h un site web proposant une interface utilisateur, permettant à ce dernier de créer ses propres histoires interactives et de visionner toutes les histoires proposées publiquement.
La gestion des chapitres et des liaisons entre ces derniers étant importante pour le comportement interactif des histoires, selon le choix du lecteur.
  • PHP PDO
  • Laravel
  • MySQL
  • HTML5
  • CSS3
  • Git

  • GitHub
Projet réalisé en collaboration avec 2 étudiants en DUT Informatique et 4 étudiants en DUT MMI.

Projet réalisé à l'IUT de Lens
Description
Au cours de la première année en DUT Informatique, j'ai réalisé en équipe un programme de jeu en python avec interface graphique (tkinter), ce projet devait avoir pour particularité de représenter un jeu 2048, mais toutefois avec une variante en mode tetris.

Projet personnel
